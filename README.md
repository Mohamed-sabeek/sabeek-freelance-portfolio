# 🚀 Sabeek Freelance Portfolio

A modern, responsive freelance portfolio built using **React + Vite + Tailwind CSS**.
This portfolio showcases my services, projects, pricing, and allows clients to contact me directly via a working contact form.

---

## 🌐 Live Demo

👉 https://sabeek-freelance-portfolio.vercel.app/

---

## ✨ Features

* ⚡ Fast & optimized (Vite)
* 🎨 Modern UI with dark + gold theme
* 📱 Fully responsive design
* 📊 Pricing section with service tiers
* 📬 Contact form with EmailJS integration
* 💬 WhatsApp direct contact option
* 🧑‍💻 Project showcase section
* ⭐ Testimonials (client feedback ready)

---

## 🛠️ Tech Stack

* **Frontend:** React 19, Vite
* **Styling:** Tailwind CSS v4
* **Routing:** React Router DOM
* **HTTP:** Axios
* **Icons:** Lucide React
* **Email Service:** EmailJS

## ⚙️ Environment Variables

Create a `.env` file in the root and add:

```
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```

## 🚀 Getting Started

### 1. Clone the repository

```
git clone https://github.com/Mohamed-sabeek/sabeek-freelance-portfolio.git
cd flportfolio
```

---

### 2. Install dependencies

```
npm install
```

---

### 3. Run the development server

```
npm run dev
```

---

### 4. Build for production

```
npm run build
```

---

## 📬 Contact Form Setup (EmailJS)

1. Create account at https://www.emailjs.com
2. Add Email Service (Gmail)
3. Create Email Template
4. Add credentials in `.env`
5. Restart the project

---

## 📦 Deployment

This project is deployed using **Vercel**.

Steps:

* Push code to GitHub
* Import project in Vercel
* Add environment variables
* Deploy
